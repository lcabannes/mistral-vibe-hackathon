Use `team_activity` to read the already-synchronized, privacy-reduced activity for this project's team workspace.

Use this to understand which coworkers and agents are active before duplicating work. The result contains only presence, branch, agent profile, coarse state, optional fixed activity summaries, and timestamps. It never provides prompts, reasoning, tool inputs or outputs, commands, paths, environment values, configuration, approvals, or response text. This tool cannot contact, control, message, approve, or cancel another agent.
